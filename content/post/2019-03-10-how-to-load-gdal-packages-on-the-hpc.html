Overview

  • system에 설치하려는 R의 특성
  • 로컬컴퓨터에서는 적용이 용이하나 클러스터에서는 적용불가. 이유는 R이 클러스터 전체환경을 바꾸려는 시도를 하기 때문.
  • 따라서 사용자 내부환경에서 R 패키지를 설치하게 되었음.

Procedure

    1. Load miniconda

      module load miniconda3-4.5.4-gcc-5.4.0-hivczbz
    1. Install R in a new environment in renv2
conda create -p renv2
conda install R

    1. Activate R on renv2

      source activate /home/hs621/renv2

    1. Install desired R packages: sf, sp, tidyverse

      conda install -p /home/hs621/renv2  R r-sf r-sp
